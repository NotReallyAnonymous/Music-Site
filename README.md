# Music Site

Local music demo library for browsing projects and WAV files.

## Clone

```bash
git clone <your-repo-url>
cd Music-Site
```

## Install dependencies

```bash
npm install
```

## Run

```bash
npm start
```

By default the server listens on `0.0.0.0:3119`. You can override with:

```bash
HOST=127.0.0.1 PORT=3000 npm start
```
